# Megasuper

Aplicacion full-stack con arquitectura en capas (MVC). Incluye logica de dominio para productos y descuentos, un servidor Express con rutas, controladores, servicios y repositorios, y logica de cliente para operaciones de carrito.

Dependencias: Express 5, Zod, CORS, dotenv.

**Iniciar:**

```bash
cd ejercicio-megasuper-2026
npm install
npm start
```

El servidor se levanta en el puerto definido en `server/.env`.
